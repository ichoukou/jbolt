Autocomplete组件
切换显示Autocomplete组件绑定的隐藏域
通过Input标签上 data-autocomplete 指定作为一个可以通过输入内容自动后台检索数据并提示 完成的组件
data-url="admin/user/autocomplete" 指定数据源地址

AutoSelect组件
data-onlyleaf="true" 设置只能选择叶子节点
通过在Select标签上配置数据源地址，已选择项目值，可实现自动加载、自动校验、自动选中，无需写JS
AutoSelect组件-自定义Value和Text需要的字段属性
通过在Select标签上配置 data-text-attr="name" data-value-attr="id" 可以设置option的text和value的取值字段
这样后端接口就不用特殊处理数据里必须有text和value这两个属性，Model也不用非得实现Option接口了
而且支持text显示多个字段的值data-text-attr="name,sex,age"
AutoSelect组件-多级联动
上级的select 基本配置跟普通自动加载的select一样，如果需要多级联动，就需要配置data-linkage="true" 这个属性就是声明这个select属于这个多级联动的一部分，额外还需配置一个data-sonid="需要联动的下级Select组件的ID。 这样就知道了切换这个select选项的时候 触发哪一个select去重新加载数据，达到联动的效果。 注意：被联动的select 也需要增加data-linkage="true" 属性，但是最好去掉data-autoload属性。 最后，就是下级被联动的select需要一个数据源地址data-srcurl=""这里的srcurl需要写好URL最后是个自动拼接数据，当上级切换选择后，选择后的value值将会结合srcurl拼接下级Select完整的数据源URL地址。
